動作環境

System Requirements

必須環境

OS Windows 10 / 11（64bit）
CPU Intel Core i3 以上 / AMD Ryzen 3 以上（推奨: Core i5 / Ryzen 5 以上）
メモリ 4GB 以上（推奨: 8GB 以上）
GPU OpenCL 1.2 対応GPU（NVIDIA / AMD / Intel 内蔵GPU）
ディスク 約500MB の空き容量（KataGo AIモデル含む）
ネットワーク インターネット接続（ネット対局・自動アップデートに必要）

AI エンジン

KataGo AIはインストーラーに同梱されています。 別途ダウンロードやインストールは不要です。インストール後すぐにAI対局や形勢判断をご利用いただけます。

GPUについて： OpenCL 1.2 対応GPUが必要です。Intel 内蔵GPU（HD Graphics 4000以降）でも動作しますが、 NVIDIA GeForce または AMD Radeon のGPUを使用すると、AI対局や形勢判断の応答速度が向上します。 初回起動時にGPUのチューニングが自動で行われます（数分かかる場合があります）。
