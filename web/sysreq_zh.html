System Requirements

必要环境

操作系统 Windows 10 / 11（64位）
CPU Intel Core i3 以上 / AMD Ryzen 3 以上（推荐: Core i5 / Ryzen 5 以上）
内存 4GB 以上（推荐: 8GB 以上）
GPU 支持OpenCL 1.2的GPU（NVIDIA / AMD / Intel 集成显卡）
磁盘空间 约500MB 可用空间（含KataGo AI模型）
网络 需要互联网连接（网络对局及自动更新所需）

支持语言

界面语言 日语、英语、中文、韩语
读秒语音 日语、英语、中文、韩语
