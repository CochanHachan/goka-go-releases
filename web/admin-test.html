管理者画面のダウンロード

対局クライアントの修正版（staging）を先に入れる場合は、修正版ZIP（対局クライアント） からダウンロードしてください。

以下の管理者 ZIP は、対局クライアントの「ダウンロード」と同様、GitHub Releases から取得します。

修正版（staging）の内容

次の ZIP を展開すると、いまの main 向けの管理者クライアント（PyInstaller ビルド）が入ります。

  • ウィンドウタイトルに [STAGING] が付きます。
  • シートの下にタブ 「サイズ・位置」（テーマ）と 「時間」（対局申込タイムアウト・Fischer・ロボ出現）があります。
  • 展開後は igo_admin\igo_admin.exe を実行してください。

修正版の取得は、ページ最上の緑枠内をクリックしてください（ZIP のダウンロードが始まります）。番号付きリリースでは build-and-deploy がこの枠内のリンク先だけ差し替えます。

手元でビルドして配布する場合

リポジトリのルートで pyinstaller igo_admin.spec --noconfirm を実行し、続けて py -3 tools/package_igo_admin_zip.py で dist/goka-admin-local.zip を生成できます。展開後は igo_admin\igo_admin.exe を実行してください。静的サイトや社内共有に ZIP を置く場合は、上の緑枠のリンク先をその URL に差し替えます。
