修正版（staging）の内容

次の ZIP を展開すると、いまの main 向けの管理者クライアント（PyInstaller ビルド）が入ります。

  • ウィンドウタイトルに [STAGING] が付きます。
  • シートの下にタブ 「サイズ・位置」（テーマ）と 「時間」（対局申込タイムアウト・Fischer・ロボ出現）があります。
  • 展開後は igo_admin\igo_admin.exe を実行してください。

修正版の取得は、ページ最上の緑枠内をクリックしてください（ZIP のダウンロードが始まります）。番号付きリリースでは build-and-deploy がこの枠内のリンク先だけ差し替えます。
